### Getting Started
- clone this repo

#### Server
- go to the server path `../packages/server`

```sh
# install dependencies
> yarn
# or
> yarn install

# copy .env file
> cp .env.example .env

# start project
> yarn start

# open in
http://localhost:9000/api/version
```
#### Web
- go to the web path `../packages/web`

```sh
# install dependencies
> yarn
# or
> yarn install

# copy .env file
> cp .env.example .env

# start project
> yarn dev
```
